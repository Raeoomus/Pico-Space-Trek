# Pico-Space-Trek
A port of the game Space Trek by Jake Commander for the TRS-80 Color Computer that appeared in Color Computer Magazine in May 1983.
https://archive.org/details/color-computer-magazine-1983-05/page/n37/mode/2up

Still very much a work in progress.
All the technical challenges should be solved not all systems working yet. A lot of nightmares untangling the GOTO hell of the original.
